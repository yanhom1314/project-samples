== Create Project
* 创建项目

	$ mkdir hs-demo
	$ cd hs-demo
	$ cabal init

== Build and Run
* 使用stack
	
	$ stack setup
	$ stack init
	$ stack build
	$ stack exec hs-demo		
* 使用cabal

	$ cabal configure
	$ cabal build
	$ cabal run
