# jquery.validate [![spm version](http://spmjs.io/badge/jquery.validate)](http://spmjs.io/package/jquery.validate)

---

An awesome spm package!

---

## Install

```
$ spm install jquery.validate --save
```

## Usage

```js
var $ = require('jquery');
require('jquery.validate')($);

$("#form").validate({});//use
```
